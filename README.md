# ArmaDion

Выживание в мире, полном зомби, в духе Project Zomboid. Разрабатывается на Godot Engine.

## Описание

ArmaDion — это изометрическая/вид сверху survival-игра в реальном времени с открытым миром, крафтом, строительством баз и сражениями с зомби.

## Основные особенности

- **Открытый мир** — процедурно генерируемая карта с зданиями, деревьями, дорогами
- **Система выживания** — здоровье, голод, жажда, стамина
- **Искусственный интеллект зомби** — патрулирование, преследование, атаки
- **Крафт и строительство** — создание предметов и возведение укреплений
- **Инвентарь** — система сбора и использования предметов
- **Сохранение игры** — загрузка и сохранение прогресса

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD | Движение |
| ЛКМ | Атака |
| E | Взаимодействие |
| I | Инвентарь |
| C | Крафт |
| Shift | Бег |

## Запуск

1. Установите [Godot Engine 4.x](https://godotengine.org/)
2. Откройте `project.godot` в Godot
3. Нажмите F5 для запуска
