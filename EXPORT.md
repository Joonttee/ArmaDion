# Экспорт игры в .exe

## Требования:

1. **Godot Engine 4.7.1** (или совместимая версия)
2. **Шаблон экспорта для Windows** (скачивается через Godot)

## Инструкция:

### 1. Скачайте шаблон экспорта

1. Откройте Godot Engine
2. Перейдите в **Project → Export...**
3. Нажмите **Add...**
4. Выберите **Windows Desktop**
5. Нажмите **Download and Install** для скачивания шаблона

### 2. Настройте экспорт

1. В окне **Export** выберите пресет **Windows Desktop**
2. Укажите путь экспорта (например, `./ArmaDion.exe`)
3. При необходимости добавьте иконку приложения

### 3. Экспортируйте

1. Нажмите **Export Project...**
2. Выберите папку для сохранения
3. Дождитесь завершения экспорта

## Альтернативный способ (через командную строку):

```bash
godot --export-release "Windows Desktop" ./ArmaDion.exe
```

## Структура экспортированной игры:

```
ArmaDion/
├── ArmaDion.exe          # Исполняемый файл
├── ArmaDion.pck          # Файл ресурсов (если не встроен)
└── user/                 # Папка сохранений (создаётся автоматически)
    └── saves/
```

## Решение проблем:

### Ошибка "No export template found"
- Скачайте шаблон экспорта через Godot (Project → Export → Add → Download)

### Ошибка "Failed to export"
- Проверьте что главная сцена назначена (project.godot → config/run_main_scene)
- Убедитесь что все скрипты компилируются без ошибок

### Игра не запускается
- Проверьте наличие всех файлов в папке экспорта
- Запустите из командной строки для просмотра ошибок
