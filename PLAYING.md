# ArmaDion - Руководство по игре

## Управление

### Основное управление:
| Клавиша | Действие |
|---------|----------|
| WASD | Движение |
| ЛКМ | Атака |
| E | Взаимодействие |
| I | Инвентарь |
| C | Крафт |
| B | Строительство |
| F | Фермерство |
| K | Навыки и черты |
| M | Мутации |
| G | Отряд |
| Tab | Карта |
| Shift | Бег |
| ESC | Пауза |

## Геймплей

### Начало игры:
1. Запустите игру
2. Выберите "Новая игра" в главном меню
3. Исследуйте мир, собирайте ресурсы
4. Создавайте предметы и стройте базу

### Выживание:
- Следите за здоровьем, голодом и жаждой
- Избегайте зомби, особенно ночью
- Собирайте еду и воду
- Создавайте укрытие для защиты

### Исследование:
- Миры генерируются случайно
- Города содержат много лута
- Деревни безопаснее
- Ищите точки интереса (больницы, полиция)

### Крафт:
- Откройте меню крафта (C)
- Выберите рецепт
- Убедитесь что есть нужные материалы
- Создайте предмет

### Строительство:
- Откройте меню строительства (B)
- Выберите тип постройки
- Разместите на карте
- Укрепите базу

### Отряд:
- Нанимайте НПС в отряд
- Отправляйте на миссии
- Назначайте задания на базе

## Советы:

1. **Начните с малого** - постройте простое укрытие
2. **Собирайте всё** - любой предмет может быть полезен
3. **Изучайте навыки** - они дают важные бонусы
4. **Торгуйте** - обменивайте ненужное на нужное
5. **Будьте осторожны ночью** - зомби становятся агрессивнее

## Системы:

### Температура:
- Носите тёплую одежду зимой
- Избегайте воды в холод
- Используйте обогреватели

### Мораль:
- Спите на кроватях
- Ешьте вкусную еду
- Избегайте стрессовых ситуаций

### Зависимости:
- Не злоупотребляйте веществами
- Ломка может быть опасной

### Болезни:
- Обрабатывайте раны
- Используйте медикаменты
- Избегайте заражённых

## Решение проблем:

### Игра не запускается:
1. Убедитесь что установлен Godot 4.x
2. Проверьте что все файлы на месте
3. Откройте project.godot в Godot

### Ошибки в консоли:
1. Проверьте пути к скриптам
2. Убедитесь что все сцены существуют
3. Проверьте автозагрузку в project.godot

### Низкая производительность:
1. Уменьшите размер мира
2. Отключите лишние эффекты
3. Закройте другие программы
